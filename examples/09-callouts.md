# Callouts

Admonition blocks via `::: note` / `::: warning` / `::: tip`. Back to the
[index](README.md).

::: note
This is a **note**. Use it for neutral, informational asides — context the
reader might want but doesn't strictly need.
:::

::: tip
This is a **tip**. Great for shortcuts and best practices. For example: press
the **Next ›** button to page through a folder of Markdown files.
:::

::: warning
This is a **warning**. Use it for things that can bite: *"Editing the watched
file in another app triggers an auto-reload."*
:::

Callouts can contain anything — lists, code, even math:

::: note
Steps:

1. Open a file
2. Edit it elsewhere
3. Watch it reload

The reload is debounced and preserves your scroll position.
:::
